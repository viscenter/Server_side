A groovlet (java servlet written in Groovy) implementing the CHS Image Extension
for CITE Collections.

To see a list of suggested ant targets, run "ant" or "ant usage".  In brief:

chsimg    Builds the CHS Image Extension in a jetty container
run-img   Runs the servlet in a jetty container 
war       Packages a war file you can drop into any servlet container.
